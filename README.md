# Kenneth Oyahebholo — Portfolio

Personal portfolio site for Kenneth Oyahebholo Izosea, Senior Frontend Engineer.

## Stack

- **Next.js 14** (Pages Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

## Running locally

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

```
pages/
  index.page.tsx      # Entry point
  _app.page.tsx       # Global CSS + font
  _document.page.tsx  # HTML document shell
components/
  NavBar.tsx
  sections/           # Hero, About, Metrics, Experience, Projects, Skills, Contact
  ui/                 # AnimatedCounter, GlassCard
styles/
  globals.css         # Liquid Glass design system
```
